# Fil Good - Calculateur d'Impression 3D

Application web moderne pour calculer les coûts d'impression 3D, gérer les filaments et générer des devis professionnels.

## Fonctionnalités

*   **Calculateur de Coût** : Estimez le coût de revient et le prix de vente en fonction de la matière, du temps machine, de la main-d'œuvre et des marges.
*   **Gestion des Filaments** : Catalogue visuel des filaments disponibles (PLA, PETG, TPU, etc.) avec filtres et tri.
*   **Générateur de Devis PDF** : Créez des devis professionnels au format PDF en un clic.
*   **Mes Créations** : Galerie pour exposer vos réalisations.
*   **Design Responsive** : Interface optimisée pour Desktop, Tablette et Mobile.

## Technologies

*   **React** (Vite)
*   **Tailwind CSS** pour le styling
*   **jspdf** & **jspdf-autotable** pour la génération de PDF
*   **Lucide React** pour les icônes

## Installation

1.  Cloner le dépôt :
    ```bash
    git clone https://github.com/Karmadibsa/Fil-Good.git
    ```
2.  Installer les dépendances :
    ```bash
    cd Fil-Good
    npm install
    ```
3.  Lancer le serveur de développement :
    ```bash
    npm run dev
    ```

## Utilisation

Remplissez les champs du calculateur avec vos paramètres (prix du filament, temps d'impression, etc.). Le coût total et le prix de vente suggéré sont mis à jour en temps réel. Cliquez sur "Générer Devis PDF" pour télécharger le document.
